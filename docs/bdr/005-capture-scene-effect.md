# BDR-005: ボール使用後の捕獲演出シーンの追加

## ステータス

Accepted

## 結論

ボール使用 (`capturing` フェーズ) から結果画面 (`result` フェーズ) の間に捕獲演出シーンを追加する。選択したボールを3回揺らすアニメーションの後、捕獲成功時は花火風、失敗（escaped）時は煙幕風の CSS エフェクトを再生してから結果画面に遷移する。

## 背景・課題

現状はボール使用ボタンを押すと即座に結果画面へ遷移し、捕獲の成否がいきなり表示される。ボールを投げてからポケモンが捕まるかどうかの間に演出を挟むことで、原作ゲームのようなドキドキ感を再現し、ゲームとしての UX を向上させる。
