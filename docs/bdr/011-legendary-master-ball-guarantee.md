# BDR-011: 伝説・幻ポケモンのマスターボール確定捕獲

## ステータス

Accepted

## 結論

伝説・幻ポケモン (`is_legendary`/`is_mythical` が true) に限り、名前当てに正解し、かつ最終評価点が70以上のとき、確定したボール種別を新設の「マスターボール」にする。マスターボールでの捕獲は確率計算をバイパスし、必ず捕獲に成功する。

- 対象となる伝説・幻の判定は、出題ポケモンの `is_legendary`/`is_mythical` を用いる。エンカウント抽選専用の `LEGENDARY_MYTHICAL_IDS` とは別系統のデータであり、両者は概ね一致するのみで一致は保証されないため、本仕様の判定には使わない。
- 条件成立時、捕獲スタンバイ画面の「マスターボールを　手に　入れた！」表示に続けて、博士のセリフ「今こそ　このボールを　使うときだ！」を吹き出しで添える。
- 条件不成立時 (最終評価点が70未満、または名前当てに失敗・スキップした場合) は既存仕様のまま (poke/great/ultra ボールによる確率捕獲)。

## 背景・課題

伝説・幻ポケモンは2段階抽選 (BDR-003) により1%でしか出現しない希少な個体であり、高スコアを出しても確率で捕獲に失敗すると、通常のポケモンより落胆が大きい。レア度の高い伝説・幻ポケモンに限り、一定のスコアを超えたら確実に捕獲できるようにしたい。
