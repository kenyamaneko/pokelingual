# BDR-008: 翻訳結果画面のリビール順序と HP・ダメージのパーセント表記

## ステータス

Accepted

## 結論

翻訳結果画面 (`guessing` フェーズ) は、博士のコメント → 日本語の説明文 → HP 減少 → ダメージ数値の順に段階的に開示する。博士のコメントと日本語の説明文はタイプライター演出で1文字ずつ表示し、その開示が終わってから HP バーと HP 数値を満タンから残量まで連動して減少させ、最後にダメージ数値とこうか表示を出す。HP とダメージの数値はどちらも分母 (100) を表示しないパーセント表記とする (例: 70/100 → 70%)。

## 背景・課題

従来の開示順序 (BDR-004: メーター → こうか表示 → 博士のコメント) では、HP ゲージが減り切る前にダメージ数値が確定表示されており、結果の意外性が失われていた。文章の開示を先に行い数値の確定を最後に回すことで、結果発表までのドキドキ感を高める。

## 詳細

- 「君の翻訳」欄は既存どおり最上部に静止表示のままとする (今回の対象外)

## Amendment: 2026-07-20 解答例と博士のコメントの開示順序を入れ替え

出題英文・君の翻訳・解答例は同じカードにまとまり、博士のコメントは別カードに分かれて配置されている。この配置のまま博士のコメントを先に開示すると、原文と解答例の表示が時間的に分断され、対応関係が読み取りづらくなっていた。

開示順序を 解答例 → 博士のコメント → HP 減少 → ダメージ数値 に変更した。
