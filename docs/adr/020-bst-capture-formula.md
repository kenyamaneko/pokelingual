# ADR-020: 種族値ベースの捕獲率計算式

## ステータス

Accepted

## 結論

強いポケモンほど捕まえにくくしてゲーム性を高めるため、捕獲率を種族値（BST）とスコアを説明変数とするシグモイド関数で計算し、名前当ての結果で決まるボール倍率を掛ける。出題文中のポケモン名は伏せ字にし、名前推測のヒントにならないようにする。強敵の捕獲に達成感が生まれ、名前推測がより挑戦的になる。

## 背景・課題

従来の捕獲率は score × nameMultiplier × guessPenalty の単純な線形計算で、全ポケモンが同じ条件で捕まった。強いポケモン（高種族値）を捕まえた達成感がなく、ゲーム性が物足りなかった。ポケモン本家では種族値が高いほど捕獲が難しく、同様の体験を翻訳ゲームでも実現したかった。

具体的な式の係数は体感値をもとにしたチューニング値であり、固定値としては本 ADR に書かない（Amendment で複数回調整している）。

## Amendment: 2026-02-14 捕獲率式の係数を再調整

捕獲率式の係数を調整した。狙いはスコア 90 + スーパーボールなら伝説ポケモン（BST 680）でもほぼ確実に捕獲できるようにすること。ボール倍率（モンスターボール 1.0x / スーパーボール 2.0x / ハイパーボール 3.0x）は変更しない。
