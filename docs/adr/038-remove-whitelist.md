# ADR-038: ホワイトリスト機構の撤去

## ステータス

Accepted

## 結論

ホワイトリスト機構を撤去し、常時公開モードにする。[ADR-022](022-google-signin-and-public-mode.md) ではホワイトリストを「空配列で公開モード」として残し、条件分岐自体は温存していたが、その分岐ごと取り除く。dev の保護は Firebase プロジェクトの分離で担保する。

## 背景・課題

prod を一般公開する方針が確定し、ホワイトリストを維持する前提自体がなくなった。リリース前監査で prod のホワイトリストが運用と乖離していたこと（空配列を想定していたが、作成時の 1 件が残留していた）も撤去の後押しになった。
