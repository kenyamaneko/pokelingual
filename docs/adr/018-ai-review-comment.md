# ADR-018: 翻訳スコアリングに AI レビューを追加

## ステータス

Accepted

## 結論

点数の理由が分かるフィードバックを与えるため、Gemini API のプロンプトにレビューフィールドを追加し、日本語レビューを返す。スコアラベル（「効果は ばつぐんだ！」等）と呼び名を分けて区別する。ユーザーは具体的な講評と難しい単語の解説を得られる。

## 背景・課題

翻訳スコアリングでは 0-100 の数値スコアのみを返していた。ユーザーが「なぜこの点数なのか」を理解する手がかりがなく、学習体験としてフィードバックが不十分だった。また、スコアラベルと AI フィードバックの両方を「コメント」と呼ぶと紛らわしかった。

レビューは日本語で 2-3 文、簡単な漢字とスペース区切りという PokeAPI の日本語説明文に近いスタイルで、翻訳の良い点・改善点・難しい英単語やフレーズの解説を、励ましつつ正直に、ポケモン博士のような口調で伝える。
