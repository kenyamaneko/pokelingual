# ADR-006: DBに Firestore を採用する

## ステータス

Accepted

## 結論

データストアには Firestore を採用する。

## 背景・課題

開発者が慣れていて手軽なことに加え、Firebase Auth・Cloud Run など他の Google Cloud サービスとの相性がいい。コストを抑えたい個人開発で、想定するデータモデルもそこまで複雑にならない見込みだった。

## 不採用案

- **Cloud SQL**：リレーショナルな表現力は得られるが、常時起動インスタンスのコストがかかる。データモデルがそこまで複雑にならない想定の下では、コストに見合わない。
